## ğŸ› ï¸ Installation

Make sure you have **Node.js (v16 or later)** and **npm** or **Yarn** installed.

### 1. Clone the Repository

```bash
git clone https://github.com/aldiposhnjari92/wynn-resort
cd wynn-resort
npm install --force
```

### 2. Install the App

```bash
npm install --force
```

or if you are using yarn
```bash
yarn install
```

### 3. Run the App
```bash
npm run dev
```
